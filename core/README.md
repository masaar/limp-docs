# Core Package
LIMP Core package is the package that provides the basic functionalities of a LIMP with number of LIMP modules, namely:
1. [Session](./session.md).
2. [User](./user.md).
3. [Group](./group.md).
4. [Setting](./setting.md).
5. [Diff](./diff.md).
6. [Realm](./realm.md).
